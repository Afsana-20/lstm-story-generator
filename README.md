# LSTM Story Generator

A deep learning project that generates story continuations using an LSTM model built in Python with TensorFlow and Gradio.

## Demo
![App Screenshot](app_screenshot (2).png)


## How it works
1. Input a story beginning
2. LSTM model predicts the next words
3. Output is displayed in the web app

## Tech Stack
- Python
- TensorFlow / Keras
- Gradio
- Google Colab

## How to Run
Open the notebook in Google Colab and run all cells top to bottom.
